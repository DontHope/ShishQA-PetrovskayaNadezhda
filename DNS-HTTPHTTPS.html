DNS- это система, содержащая IP адреса/ домены (буквенное имя) сервера.
HTTP- это протокол передачи данных между пользователем и сервером
HTTPS-это тот же протокол передачи данных между пользователем и сервером, но защищенный с
помощью шифра.
Различие: -HTTPS расширенный протокол http с защитой передаваемых данных
-По-разному отображаются в адресной строке,
-Система с протоколами http оповещают пользователя о небезопасности сайта
Версии: HTTP 1.1 При передачи различных элементов создаются несколько соединение, следовательно
нужно делать совершаться дополнительные действия/запросы для отображения полной информации
HTTP 2 При одном запросе выдает полный объем информации. Следовательно сокращает время открытия
того или иного сайта
Заголовки в адресной строке сокращаются у HTTP 2
