1. Общие сведения: 
Название продукта: Hardload Application
Версия:  1.0
Размер: 32.0 КБ
Назначение: копирование файла из одного места в другое
Условия использования ПО: 
-ОС: Windows XP,  Windows 7, Windows 8,  Windows 8.1, ,  Windows 10
-32.0 КБ свободного пространства на жёстком диске
Назначение ПО:  осуществляет копирование файлов в любое локальное место 

2. Подготовка к работе
Открыть программу Hardload

3. Работа с системой







Знакомство с программой




Для копирования файла из одной папки в другую необходимо:
1.Нажать на  «…» 
 
2. В открытом окне выбрать необходимый файл (в нашем примере формат jpg) и нажать кнопку «открыть»

3. Файл появится в программе 

4. Нажать поставить галочку в окошко рядом с названием файла 

5. Нажать на «…» внизу программы и выбрать папку, куда будет скопирован файл 

6. Нажать «начать грузить» 

8. Файл загружен. Можно проверить, зайдя в указанную папку 


Удаление файла 
1. Повторит шаг 1-4 (Для копирования файла из одной папки в другую необходимо)
2 Нажать на файл, чтобы он выделился
. 
3. Нажать на крестик и в сплывшем окне выбрать «да»

4. Ограничения
1. Добавление нескольких файлов одновременно  в список невозможно
2. Нет возможности добавить файл, не используя «…»
3. Если добавленный файл в список, удалить из места его нахождения, то скопировать данный файл не будет возможности
4. Нет возможности настроить размер окна приложения по своему желанию
5. Осуществить копирование файла на веб-сайт невозможно
